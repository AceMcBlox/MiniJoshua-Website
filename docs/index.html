Getting Started Last updated: 2020-01-12

This will cover inviting MiniJoshua, Setting up MiniJoshua and The Basics.





Inviting the Bot

To invite the bot click the "Add to Discord" button in the top right corner.

You should then see a prompt where you need to select which server to add the bot to.

After you have approved and added the bot you should see it in your discord server.
